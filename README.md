# 深圳中考单词冲刺 · Vocab Quest

一个面向深圳中考英语薄弱学生的纯静态背单词网页，适合部署到 GitHub Pages。学习数据保存在浏览器 `localStorage`，适合单人使用。

## 页面

- 背诵页：单词卡片、今日进度、总掌握率、金币、闯关地图、遗忘曲线。
- 管理员看板：总进度、今日完成率、高风险词、掌握分布、7 天趋势、干预建议。
- 设置页：每日目标、任务策略、报告导出、演示数据重置。
- 产品需求：从 PM 视角梳理目标用户、核心问题、关键指标、功能优先级和部署方案。

## 手机 / 微信使用

- 首屏和操作按钮已按手机竖屏优化，底部固定导航适合拇指操作。
- 可以把 GitHub Pages 链接发到微信，学生收藏后每天打开使用。
- 数据存在当前浏览器里，换手机或清缓存会丢失；需要长期跨设备使用时再接入后端。

## GitHub Pages 部署

1. 新建或使用 `https://github.com/Paden79555/words_recite_lele.git`，把本目录里的文件提交到仓库根目录。
2. 进入仓库 Settings → Pages。
3. Source 选择 Deploy from a branch。
4. Branch 选择 `main`，目录选择 `/root`。
5. 保存后等待 GitHub 生成访问地址。
